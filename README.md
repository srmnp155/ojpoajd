# Flask CSV Extracter

A simple Flask application to extract a specified column from an uploaded CSV file and download it as a new CSV.

## Usage

1. Install dependencies:

```
pip install -r requirements.txt
```

2. Run the app:

```
python app.py
```

3. Open your browser at `http://127.0.0.1:5000/` and upload a CSV file, specify the column to extract, and download the extracted CSV.
